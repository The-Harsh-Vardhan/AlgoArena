## ✨ Features

### 🚀 Core Capabilities

- **🔄 Dynamic Analysis**: Upload any tabular dataset for instant ML analysis
- **🧠 Auto Problem Detection**: Automatically identifies classification vs regression problems
- **🛠️ Smart Preprocessing**: Intelligent handling of missing values, categorical encoding, and feature scaling
- **🤖 Multiple Algorithms**: Comprehensive suite of ML algorithms from classical to advanced
- **📊 Interactive Visualizations**: Rich, interactive charts and plots using Plotly
- **📈 Performance Metrics**: Detailed accuracy, precision, recall, F1-score analysis
- **💾 Export Results**: Download analysis results and visualizations
- **🎯 Real-time Processing**: Instant feedback and results

### 🧮 Supported Algorithms

#### Classification Algorithms
- **Logistic Regression** - Linear probabilistic classifier
- **Random Forest** - Ensemble method with decision trees
- **Support Vector Machine (SVM)** - Kernel-based classifier
- **Gradient Boosting** - Sequential ensemble method
- **XGBoost** - Optimized gradient boosting
- **LightGBM** - Fast gradient boosting framework
- **CatBoost** - Categorical feature boosting

#### Regression Algorithms
- **Linear Regression** - Classic linear approach
- **Random Forest Regressor** - Ensemble regression
- **Support Vector Regression (SVR)** - Kernel-based regression
- **Gradient Boosting Regressor** - Sequential ensemble
- **XGBoost Regressor** - Optimized boosting for regression
- **LightGBM Regressor** - Fast gradient boosting
- **CatBoost Regressor** - Categorical boosting for regression

### 📊 Visualization Features

- **Model Performance Comparison**: Side-by-side algorithm comparison
- **Feature Importance Charts**: Understand which features matter most
- **Confusion Matrices**: Detailed classification performance
- **ROC Curves**: Receiver Operating Characteristic analysis
- **Distribution Plots**: Data distribution visualization
- **Correlation Heatmaps**: Feature relationship analysis

## 🛠️ Installation

### Prerequisites

- Python 3.8 or higher
- pip (Python package installer)

### Quick Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/The-Harsh-Vardhan/AlgoArena.git
   cd AlgoArena
   ```

2. **Install dependencies**
   ```bash
   pip install -r streamlit_app/requirements.txt
   ```

3. **Run the application**
   ```bash
   streamlit run streamlit_app/app.py
   ```

4. **Open in browser**
   ```
   Navigate to: http://localhost:8501
   ```

### Alternative Setup (Using batch script)

For Windows users, we provide a convenient batch script:

```bash
# Run the batch script for automatic setup
setup_and_run.bat
```

### Docker Installation (Coming Soon)

```bash
# Docker support coming soon
docker pull algoarena/ml-analyzer
docker run -p 8501:8501 algoarena/ml-analyzer
```
